# flinq (backend)

Python 3.13 + FastAPI + SQLAlchemy 2.x async + Taskiq backend for Flinq.

See [ADR-0006](../docs/adr/ADR-0006-tech-stack.md) for the full tech stack rationale.

## Layout

```
backend/
├── pyproject.toml
├── uv.lock              # generated by `uv sync`
├── alembic.ini
├── Dockerfile           # multi-stage: builder, frontend-build, dev, runtime
├── src/flinq/
│   ├── main.py          # FastAPI app factory
│   ├── core/            # config, logging, db — cross-cutting concerns
│   ├── api/             # FastAPI routers (health, per-module routers)
│   ├── modules/         # domain modules (see architecture overview §7)
│   ├── worker/          # Taskiq broker + task definitions
│   └── cli/             # typer admin commands
├── migrations/          # Alembic migrations
└── tests/
```

## Local development

```bash
# Install uv: https://github.com/astral-sh/uv
uv sync

# Run API in dev mode
uv run flinq serve

# Run Taskiq worker
uv run flinq worker

# Run tests
uv run pytest

# Lint & format
uv run ruff check .
uv run ruff format .

# Type check
uv run pyright

# Database migrations
uv run alembic upgrade head
uv run alembic revision --autogenerate -m "describe change"
```
